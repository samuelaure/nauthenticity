# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-03-19
### Added
- **Intelligence Engine**: Vectorized similarity search using PGVector and OpenAI `text-embedding-3-small`.
- **Strategy Extraction**: Automated hook, pillar, and CTA extraction using GPT-4o.
- **Semantic Search API**: New `POST /api/search` endpoint for natural language content discovery.
- **Data Backfill**: Scripts for migrating legacy transcripts into the vector brain.

## [1.0.1] - 2026-03-18
- Initial release with Audit Remediation and Hardening.
